Student Management System
A simple web-based application for managing student records, built using Java, Spring Boot, and Thymeleaf. This project provides a basic CRUD (Create, Read, Update, Delete) functionality to manage student information, including names, emails, and courses.
Features

Add new students with their details (name, email, course).
View a list of all registered students.
Delete existing student records.
Responsive web interface using Thymeleaf templates.
RESTful API endpoints for programmatic access.
Integrated with an in-memory H2 database for data persistence.
Swagger UI for API documentation.

Technologies Used

Java: Core programming language.
Spring Boot: Framework for building the application with auto-configuration and dependency management.
Thymeleaf: Server-side templating engine for dynamic HTML rendering.
Spring Data JPA: For database operations with Hibernate as the JPA provider.
H2 Database: In-memory database for development and testing.
Springdoc OpenAPI: For generating API documentation (Swagger UI).
